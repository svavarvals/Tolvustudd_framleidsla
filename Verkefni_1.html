Vefsíða

Verkefni 1 í Tölvustuddri Framleiðslu - Vefsíðugerð

Yfirlit yfir fyrsta Verkefnið

Fyrsta verkefni námskeiðsins var að hanna og setja upp vefsíðu sem safnar saman verkefnum námskeiðsins og þjálfar notkun á Git. Vefsíðan veitir innsýn í sköpunarferlið, útskýrir framkvæmd skref fyrir skref og lýsir þeim áskorunum sem hafa komið upp á vegferðinni.

Undirbúningur

Áður en ég byrjaði á vefsíðunni fór ég vandlega yfir verkefnalýsinguna. Ég nýtti mér myndböndin á YouTube frá Hafliða, þau voru mjög gagnleg til að koma manni af stað.

Með skýra hugmynd um útlit vefsíðunnar hóf ég niðurhal á þeim verkfærum sem ráðlagt var að nota fyrir verkefnið. Ég setti upp Brackets, einfaldan og notendavænan kóðaritil með forskoðun í rauntíma, sem reyndist lykilatriði í þróunarferlinu. Einnig stofnaði ég GitHub reikning til að hýsa vefsíðuna og setti upp Git Bash frá Git til að hlaða verkefnaskrám í geymsluna.

Aðlögun og uppfærslur á vefsíðunni.

Eftir að undirbúningsferlinu lauk lagði ég áherslu á að velja hentugt vefsíðusniðmát frá HTML5UP. Markmiðið var að finna einfalt, sjónrænt aðlaðandi og hagnýtt sniðmát með fellivalmynd sem auðveldaði skipulag og aðgengi að verkefnum. Ég ákvað að nota Spectral sniðmátið, sem uppfyllti þessi skilyrði fullkomlega. Ég sótti sniðmátið og hóf að sérsníða það með Brackets.

Ég fjarlægði óþarfa hluta af forsíðunni og bætti við viðeigandi þáttum til að uppfylla kröfur verkefnisins. Útbjó sérstakar HTML skrár fyrir hverja undirsíðu og tengdi þær við fellivalmyndina á réttan hátt. Endurnýjaði allan staðsetningartexta á forsíðunni og í valmyndinni til að endurspegla innihald og tilgang síðunnar.

Uppsetning á GitHub

Ég bjó til repository á GitHubinu mínu sem heitir Tolvustudd_framleidsla og ég setti það á Public. Næst opnaði ég Git Bash, sem ég hafði sett upp, beint í möppu vefsíðunnar. Til að gera þetta hægrismellti ég inni í möppu vefsíðunnar og valdi „Open Git Bash here“.

Skref 1: Frumstilling á Git Repository

Þetta setur upp Git tracking fyrir möppuna.


									git init

Skref 2: Tengja Repository við GitHub


									git remote add origin https://github.com/svavarvals/Tolvustudd_framleidsla

Til að staðfesta hlekkinn


									git remote -v

Skref 3: Flokka skrár


									git add .

Ég fékk fyrst villu hér, það skiptir miklu máli að það sé bil á milli add og punktsins. Þá ætti þessi skipun að virka.

Skref 4: Committa skrárnar


									git commit -m "Initial commit: Upload website files"

Skref 5: Ýta skránum á GitHub


									git push -u origin master

Forsíðumyndin

Fyrst var bara blörruð mynd sem forsíðumynd en með þessum kóða að neðan getur maður sett sýna eigin mynd sem forsíðumynd á vefsíðunni.

Væntingar mínar til áfangans

Markmið mitt með þessum áfanga er að læra að skipuleggja og vinna krefjandi og vönduð verkefni, sem ég get kynnt á faglegan og áhrifaríkan hátt hér á síðunni. Ég vil leggja metnað í að skila verkefnum af mikilli nákvæmni og skýra vel frá þeim, þannig að þessi síða verði dýrmæt eign fyrir mig í framtíðinni og sýni styrkleika mína á markvissan hátt.
